# 🚕 Ola Ride Booking & Cancellation Analysis

## 📌 Project Overview
This project analyzes ride booking and cancellation patterns to understand operational inefficiencies and customer behavior.

## 🛠 Tools Used
- SQL
- MS Excel

## 📊 Dataset Description
The dataset includes:
- Booking ID
- Ride type
- Payment method
- Cancellation reason
- Time and location details

## 🔎 Analysis Performed
- Cancellation rate calculation
- Driver vs customer cancellation comparison
- Peak booking hour analysis
- Payment mode trend analysis

## 📈 Key Insights
- Identified peak cancellation time slots
- Higher cancellation rate in specific ride types
- Payment mode impact on booking completion
- Demand surge patterns during specific hours
